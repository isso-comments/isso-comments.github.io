Dependencies¶

Isso uses some of the following dependencies:

  • werkzeug – WSGI toolkit

  • itsdangerous – store signed data on untrusted clients

  • misaka – fast Markdown processor written in C

  • html5lib – HTML(5) parser and sanitizer
